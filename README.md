## Getting Started

### Prerequisites

You may have [Node.js](https://nodejs.org/en/) installed.

### Installing

Run:

```
npm install
```


## Automation



### Lint

Run:

```
npm run lint
```

## Deployment

Run:

```
npm run serve
```

## Build

Run:

```
npm run build
```
